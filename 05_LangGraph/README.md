## Why LangGraph?

Traditional agents:
❌ Hard to control
❌ Forget state
❌ Cannot manage workflows

LangGraph introduces Graph-based execution.

---

## Core Concepts

### Nodes

Single task execution.

### Edges

Flow control between tasks.

### State

Shared memory across agents.

Learn:
https://langchain-ai.github.io/langgraph/
