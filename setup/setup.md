# 🛠 Setup Guide — LangChain & LangGraph Masterclass

Follow these steps to run projects locally.

---

## 1️⃣ Install Python

Download Python 3.10+

https://www.python.org/downloads/

Check installation:

python --version

---

## 2️⃣ Clone Repository

git clone https://github.com/devsavaj/LangChain-LangGraph-Masterclass.git
cd LangChain-LangGraph-Masterclass

---

## 3️⃣ Create Virtual Environment

### Windows
python -m venv venv
venv\Scripts\activate

### Mac/Linux
python3 -m venv venv
source venv/bin/activate

---

## 4️⃣ Install Dependencies

pip install -r requirements.txt

---

## 5️⃣ Add OpenAI API Key

Create `.env` file:

OPENAI_API_KEY=your_api_key_here

---

## ✅ Setup Complete
